# FINE$$E Terminal v2.6 🚀

**Якщо ви торгуєте крипто на Solana, цей проект для вас!**

Це красива платформа для торгівлі з графіками, редактором смарт-контрактів та інтерактивною панеллю. Усе прямо у браузері, без установки ніяких програм.

## ✨ Можливості

### 🎨 **Красивий дизайн**
- **Курсор на вибір** - три стилі: Крапка, Сканер, Привид
- **Темна тема** - сучасний дизайн із синіми кольорами
- **Плавні анімації** - все рухається гарно
- **Працює на комп'ютері** - оптимізовано для великих екранів

### 🔐 **Вхід у систему**
- **Екран завантаження** - матричні символи, як у фільмі 
- **Форма входу** - гарне вікно для логіну/реєстрації
- **Запам'ятовування** - простий механізм збереження сеансу

### 📊 **Головна панель**
- **Слідкує за монетами** - SHIB, DOGE, PEPE, BONK, FLOKI в реальному часі
- **Живі графіки** - отримує дані з TradingView
- **Інформація про мережу** - показує статус, швидкість мережі (TPS), затримку
- **Редактор коду** - можна дивитися на Rust код Solana контрактів

### 🛠️ **Для розробників**
- **Відредагуйте контракти** - синтаксис для Rust підсвічується
- **Легко змінювати** - HTML, CSS та JavaScript розділені в окремих файлах
- **Скрізь коментарі** - кожна функція пояснена

## 🚀 Встановлення

### Що потрібно

- Браузер (Chrome, Firefox, Safari, Edge) - будь-який новий
- Нічого більше не потрібно встановлювати!

### Як почати (три способи)

#### 🟢 Спосіб 1: Найпростіший

1. Перейдіть на https://github.com/mrdbv/finesse-terminal
2. Натисніть **"Code"** (зелена кнопка)
3. **"Download ZIP"** - зберегти на комп'ютері
4. Розпакуйте архів
5. **Двічі клацніть на `index.html`**

**Готово!** Відкриється в браузері. Це все! 🎉

#### 🟡 Спосіб 2: Через Git (для розробників)

```bash
git clone https://github.com/mrdbv/finesse-terminal.git
cd finesse-terminal

# Запустіть локальний сервер
python -m http.server 8000

# Перейдіть на http://localhost:8000
```

#### 🔵 Спосіб 3: Docker

Якщо у вас Docker:
```bash
docker build -t finesse-terminal .
docker run -p 8000:80 finesse-terminal
```

## 📖 Користування

### Головна сторінка

Спочатку ви бачите красиву головну сторінку з ефектом матриці. Тут можна:
- Натиснути **"Вхід"** або **"Реєстрація"** для входу
- **Будь-яке ім'я/пароль** - це тільки демо
- Змінити **стиль курсора** на кнопці "Style"
- Ввімкнути **власний курсор** на кнопці "Mode"

### Панель торгівлі

Після входу ви бачите основну панель:

1. **Виберіть монету** - клацніть на будь-яку в лівій панелі (SHIB, DOGE та інші)
2. **Дивіться графік** - праворуч з'явиться жив графік з TradingView
3. **Інформація про мережу** - в лівій панелі все про мережу
4. **Код контракту** - вгорі можна дивитися на Solana код
5. **Вихід** - клацніть логаут щоб повернутися на початок

## 🏗️ Як це влаштовано

### Послідовність сторінок

Коли ви открите програму, вона показує їх по порядку:

```
Екран завантаження (2.5 секунди)
             ↓
Головна сторінка
             ↓
Форма входу/реєстрації
             ↓
Панель торгівлі (основна сторінка)
             ↓
Кнопка вихід → Повертаєтесь на початок
```

### Що з чого робиться

| Частина | Що робить |
|---------|-----------|
| **Екран завантаження** | Матричні символи падають, потім програма запускається |
| **Форма входу** | Ввід імені та пароля з красивою анімацією |
| **Головна сторінка** | Де торгуєте - монети, графіки, інформація |
| **Панель торгівлі** | Все те, що вам потрібно для торгівлі |

## 🎯 Компоненти

### 1. Екран завантаження

Перше що ви бачите:
- Падають японські символи (як в Matrix 🤖)
- Блискавка пульсує посередині
- Текст мигає странно (glitch ефект)
- Прогрес-бар повільно заповняється
- Після 2.5 секунд все зникає

### 2. Форма входу

Красиво оформлена форма з:
- Полями для імені та пароля
- Прогрес-баром під час входу
- Гарною анімацією появи

### 3. Головна панель

Поділена на три частини:

**Ліва панель (350px):**
- Список всіх монет з цінами
- Відсоток зміни (+/-)
- Статус мережі та інформація

**Права панель:**
- Вгорі: Код Solana контракту
- Внизу: Графік з TradingView

### 4. Свій курсор

Три варіанти:
- **Крапка** - простий світлий точка
- **Сканер** - хрест з кільцем (як в грі)
- **Привид** - світне коло з розмиттям

Можна вмикати/вимикати всередину програми!

## 🎨 Кольори

Весь дизайн побудований на кількох кольорах:

```css
--bg-color: #0B0B0B        ← Чорний фон
--accent-color: #00D1FF    ← Блакитний акцент
--text-white: #FFFFFF      ← Білий текст
--text-dim: #A0A0A0       ← Сірий текст

--positive-color: #00ff88  ← Зелений (+ ціна)
--negative-color: #ff4444  ← Червоний (- ціна)
```

## 🔧 Як налаштувати

### Змініть кольори

Відкрийте `styles.css` і знайдіть верх. Там `--accent-color`. Змініть на свій!

```css
:root {
    --accent-color: #00D1FF;  ← Змініть це число
}
```

**Приклади:**
- `#FF0000` - червоний
- `#00FF00` - зелений
- `#FFFF00` - жовтий

### Додайте нові монети

В `index.html` знайдіть блок `coin-list` та додайте:

```html
<div class="coin-item" onclick="selectCoin(this, 'SYMBOL')">
    <div class="coin-name">ІМ'Я</div>
    <div class="coin-price">$0.00</div>
    <div class="coin-change positive">+0%</div>
</div>
```

Приклад - Bitcoin:
```html
<div class="coin-item" onclick="selectCoin(this, 'BTC')">
    <div class="coin-name">BTC</div>
    <div class="coin-price">$45000</div>
    <div class="coin-change positive">+2.5%</div>
</div>
```

### Оновіть графіки

У `script.js` знайдіть рядок з TradingView і змініть символ.
